       跨境电商出口申报系统旨在满足海关对跨境电子商务出口（包括跨境电商企业对个人、跨境电企业对企业商）业务的监管、统计分析等需求，为企业提供一站式办理服务。该系统包括12种单证类型，分别是“订单”、“收款单”、“运单”、“运抵单”、“离境单”、“清单”、“撤销申请单”、“清单总分单”、“退货申请单”、“退货理货明细单”、“汇总申请单”、“汇总结果单”。主要实现以上单据的录入、修改、删除、申报、查询等操作。
